# Poke Info (Aún en desarrollo)

## Descripción

Poke Info es una aplicación construida con Next.js que muestra una manera de gestionar tanto server state como client state siguiendo buenas prácticas.

El proyecto integra:

- TanStack Query (React query): Gestión de estado remoto, caché y manejo de datos asíncronos.
- Zustand: Manejo de estado global del cliente.
- PokeAPI

El objetivo es mostrar como conviven ambas herramientas dentro de la misma arquitectura, manteniendo responsabilidades bien separadas entre server state y client state.
